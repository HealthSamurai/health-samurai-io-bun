# FHIR at Scale

This recorded meetup addressed critical performance and scalability challenges when implementing FHIR systems at enterprise scale. Experts from Google, Health Samurai, Microsoft, and Zus Health discussed how to overcome the performance gaps of the FHIR REST API through practical solutions including bulk import optimization, streaming EMR data conversion, FHIR search performance enhancement, and storage architecture decisions.

## Speakers

- **Nikolai Ryzhikov** – CTO at Health Samurai
- **Vivian Neilley** – Product Manager for Google Cloud's Healthcare Data Engine
- **Sonya Huang** – Technologist at Zus Health
- **Brendan Kowitz** – Software Engineering Manager at Microsoft

## Topics Covered

- Chained search performance
- HL7v2-to-FHIR integration patterns
- Benchmarking methodologies
- Handling usability limitations in REST-based healthcare data access
