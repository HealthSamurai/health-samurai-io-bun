# Fine-grained Security Policies Beyond OAuth2

The meetup examined the challenges of fine-grained security in FHIR, exploring solutions beyond OAuth 2.0. Participants explored OAuth 2.0 limitations, how advanced access control mechanisms strengthen healthcare data security and compliance, plus implementation strategies and tools for complex modern healthcare applications.

## Speakers

- **Nikolai Ryzhikov** – CTO at Health Samurai, topic: "Access control in Aidbox"
- **Chris Grenz** – Healthcare Data Product Fanatic, topic: "Classification and Locality"
- **Michael Hansen** – Senior Director at Microsoft, topic: "Parameterized compartments"
- **Justin Richer** – CTO, Identity, Security, Standards, topic: "XYZ"
