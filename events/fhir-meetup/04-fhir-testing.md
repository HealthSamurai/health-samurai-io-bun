# FHIR Testing

This meetup covers key approaches and tools for testing FHIR implementations, including validation, conformance testing, and integration testing strategies.

## Speakers

- **Rob Scanlon** – Software Engineer at MITRE
  - Topic: "FHIR Testing with Inferno" – the open-source Inferno testing tool and its approach to validating FHIR APIs

- **Richard Ettema** – Lead Consultant at AEGIS.net, Inc.
  - Topic: "FHIR Testing Framework, TestScript and Touchstone" – the FHIR Testing Framework specification and Touchstone Testing Platform

- **Yana Beda** – Health IT Consultant, CEO at Beda Software
  - Topic: "Stresty for testing FHIR API" – testing infrastructure designed for FHIR REST API scenarios with visual test case management
