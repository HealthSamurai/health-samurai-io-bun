# FHIR Profiling

This webinar addresses key challenges healthcare organizations encounter when creating and managing FHIR profiles. Experts explore how profiling customizes FHIR resources for specific use cases while preserving interoperability, and discuss available tools and best practices for streamlining the profiling process.

- **Mark Kramer** – Chief Engineer, MITRE's Health Innovation Center
  - Topic: "Creating an Implementation Guide with FHIR Shorthand"
