# Using SQL on FHIR in the HL7 FHIR IG Publisher

This presentation will describe how the HL7 FHIR IG publisher uses SQL on FHIR ViewDefinitions to make it easier to include content out of the resources in pages in the implementation guide, and show some examples of the kinds of things that can be done with this technique.
