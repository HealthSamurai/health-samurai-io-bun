# SQL on FHIR in PostgreSQL

Nikolai will introduce two types of SQL on FHIR View Runners - ETL (in-memory) & ELT (in-database) and demo the ELT implementation for PostgreSQL using Aidbox, open-source ViewDefinition Builder, and Grafana.
