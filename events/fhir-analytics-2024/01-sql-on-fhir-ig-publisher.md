# Using SQL on FHIR in the HL7 FHIR IG Publisher

The official HL7 FHIR IG publisher already has a built-in ViewDefinition runner, introduced by Grahame. It can be used to make it easier to generate content on spec pages with SQL queries. This runner is part of the open-source HAPI framework.
