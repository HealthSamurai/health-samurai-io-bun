# FlatQuack: FHIR resources to SQL tables with DuckDB

FlatQuack is a new, open source, View Definition ELT runner that targets DuckDB, a lightweight, high performance database engine. By compiling to SQL, FlatQuack can be easily integrated into existing data pipelines that use off-the-shelf orchestration tools like DBT and Apache Airflow.
