# Aidbox Product Updates | August 2023

This 40-minute webinar covers the latest Aidbox enhancements with interactive Q&A.

## Speakers

- **Mike Kulakov** – Product Manager
- **Maxim Putintsev** – Full-stack Software Engineer, Integrations
- **Vasiliy Kupryakov** – Full-stack Software Engineer, Data API
- **Vladislav Ganshin** – Full-stack Software Engineer, Security and Access Control / Ops
- **Vitaly Banchenko** – Full-stack Software Engineer, Aidbox Forms

## Topics Covered

- **SQL on FHIR Engine** – flatten nested data and run complex queries
- **Topic-based Subscriptions** – for FHIR R4B and R5
- **BALP Support** – for CRUD and Search operations
- **Aidbox Forms Accessibility** – basic accessibility features
- **Aidbox LTS 2306** – long-term support for enterprise systems

## Target Audience

Tech leaders, architects, and healthcare IT engineers.
