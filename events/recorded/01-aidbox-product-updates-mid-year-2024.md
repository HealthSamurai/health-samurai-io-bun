# Aidbox Product Updates | Mid-Year Results 2024

This recorded webinar presents Aidbox's mid-year product developments. Mike Kulakov guides viewers through leveraging their Aidbox instance effectively while showcasing tools to enhance development with FHIR-first capabilities.

## Topics Covered

- **SQL on FHIR** – query and transform FHIR data using SQL
- **Label-based Access Control** – fine-grained security policies
- **FHIR Schema Validation** – validate resources against profiles
- **Loading FHIR Implementation Guides** – multiple approaches
- **SearchParameters & Profiling** – customize FHIR artifacts
- **Refreshed Aidbox Interface** – improved UI/UX
- **C-CDA/FHIR Converter** – bidirectional conversion improvements
- **Aidbox Forms Module** – enhanced form building capabilities

## Target Audience

Software architects, engineers, CTOs, CIOs, and professionals developing digital health solutions.
